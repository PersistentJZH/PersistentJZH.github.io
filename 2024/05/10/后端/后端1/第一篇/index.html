要使用 Yamux，你需要在 Go 项目中导入 github.com/hashicorp/yamux 包，并按照以下步骤使用 Yamux：

  1. 创建 TCP 连接：使用 Go 的 net 包中的 Dial 函数或 Listen 函数创建与远程服务器或本地监听地址的 TCP 连接。

  2. 创建 Yamux 会话：使用 yamux.Client 函数或 yamux.Server 函数创建一个 Yamux 会话。如果你是客户端，使用 yamux.Client 函数并传入已建立的 TCP 连接；如果你是服务器端，使用 yamux.Server 函数并传入已接受的 TCP 连接。

  3. 打开流：使用 OpenStream 方法在 Yamux 会话中打开一个新的流。你可以在客户端和服务器端都使用 OpenStream 方法来创建流。

  4. 发送和接收数据：使用打开的流进行数据的发送和接收。你可以使用 io.Reader 和 io.Writer 接口来读取和写入数据。

  5. 关闭流和会话：在完成数据传输后，记得关闭打开的流和会话。你可以使用 Close 方法关闭流，使用 Close 方法关闭会话。
