要使用 Yamux，你需要在 Go 项目中导入 `github.com/hashicorp/yamux` 包，并按照以下步骤使用 Yamux：

1. 创建 TCP 连接：使用 Go 的 `net` 包中的 `Dial` 函数或 `Listen` 函数创建与远程服务器或本地监听地址的 TCP 连接。

2. 创建 Yamux 会话：使用 `yamux.Client` 函数或 `yamux.Server` 函数创建一个 Yamux 会话。如果你是客户端，使用 `yamux.Client` 函数并传入已建立的 TCP 连接；如果你是服务器端，使用 `yamux.Server` 函数并传入已接受的 TCP 连接。

3. 打开流：使用 `OpenStream` 方法在 Yamux 会话中打开一个新的流。你可以在客户端和服务器端都使用 `OpenStream` 方法来创建流。

4. 发送和接收数据：使用打开的流进行数据的发送和接收。你可以使用 `io.Reader` 和 `io.Writer` 接口来读取和写入数据。

5. 关闭流和会话：在完成数据传输后，记得关闭打开的流和会话。你可以使用 `Close` 方法关闭流，使用 `Close` 方法关闭会话。

下面是一个简单的示例，演示了如何使用 Yamux 创建客户端和服务器端的会话，并在它们之间传输数据：

服务器端代码 (`server.go`)：

```go
package main

import (
	"fmt"
	"io"
	"log"
	"net"

	"github.com/hashicorp/yamux"
)

func handleClient(sess *yamux.Session) {
	// 打开一个新的流
	stream, err := sess.AcceptStream()
	if err != nil {
		log.Printf("Failed to accept stream: %s", err)
		return
	}
	defer stream.Close()

	// 从流中读取数据
	buf := make([]byte, 1024)
	n, err := stream.Read(buf)
	if err != nil {
		log.Printf("Failed to read from stream: %s", err)
		return
	}

	// 打印接收到的数据
	fmt.Printf("Received data: %s\n", string(buf[:n]))
}

func main() {
	ln, err := net.Listen("tcp", "localhost:8080")
	if err != nil {
		log.Fatalf("Failed to listen: %s", err)
	}

	for {
		conn, err := ln.Accept()
		if err != nil {
			log.Printf("Failed to accept connection: %s", err)
			continue
		}

		// 创建一个新的 Yamux 会话
		sess, err := yamux.Server(conn, nil)
		if err != nil {
			log.Printf("Failed to create session: %s", err)
			continue
		}

		go handleClient(sess)
	}
}
```

客户端代码 (`client.go`)：

```go
package main

import (
	"log"
	"net"

	"github.com/hashicorp/yamux"
)

func main() {
	conn, err := net.Dial("tcp", "localhost:8080")
	if err != nil {
		log.Fatalf("Failed to connect: %s", err)
	}

	// 创建一个新的 Yamux 会话
	sess, err := yamux.Client(conn, nil)
	if err != nil {
		log.Fatalf("Failed to create session: %s", err)
	}

	// 打开一个新的流
	stream, err := sess.OpenStream()
	if err != nil {
		log.Printf("Failed to open stream: %s", err)
		return
	}
	defer stream.Close()

	// 发送数据到服务器
	data := []byte("Hello, server!")
	_, err = stream.Write(data)
	if err != nil {
		log.Printf("Failed to write to stream: %s", err)
		return
	}
}
```

请注意，上述代码仅为示例，可能需要根据你的实际需求进行修改和调整。确保替换代码中的地址和端口为实际的服务器地址和端口。

编译和运行代码时，分别在服务器和客户端的终端中运行以下命令：

```shell
go build server.go
./server

go build client.go
./client
```

这样，客户端将向服务器发送数据，并在服务器端打印接收到的数据。

	这只是一个简单的示例，你可以根据需要进行更复杂的数据传输和处理。确保在实际应用中进行适当的错误处理和资源清理。
